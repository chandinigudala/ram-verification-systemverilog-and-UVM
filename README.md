# RAM using Verilog and SystemVerilog

## Overview

This project implements a Single-Port RAM using Verilog HDL and verifies its functionality using SystemVerilog.

---

## Features

- Read Operation
- Write Operation
- Address Decoding
- Memory Initialization
- Parameterized Memory Size
